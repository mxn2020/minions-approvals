# Welcome to Minions Approvals

We're excited to introduce **Minions Approvals** — Approval requests, human decisions, and immutable audit logs.

## What's Included

- **TypeScript SDK**: `npm install @minions-approvals/sdk`
- **Python SDK**: `pip install minions-approvals`
- **CLI**: `npm install -g @minions-approvals/cli`

## Getting Started

Check out the [documentation](https://approvals.minions.help) for a full guide.
