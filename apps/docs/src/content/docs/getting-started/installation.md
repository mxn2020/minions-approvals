---
title: Installation
description: Install Minions Approvals packages
---

## TypeScript / Node.js

```bash
npm install @minions-approvals/sdk minions-sdk
```

## Python

```bash
pip install minions-approvals
```

## CLI

```bash
npm install -g @minions-approvals/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
