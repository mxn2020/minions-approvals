# Minions Approvals

Approval requests, human decisions, and immutable audit logs

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-approvals/sdk`) — Core library for Node.js
- **Python SDK** (`minions-approvals`) — Python bindings
- **CLI Tool** (`@minions-approvals/cli`) — Command-line interface
- **Web App** — Interactive playground at [approvals.minions.wtf](https://approvals.minions.wtf)
